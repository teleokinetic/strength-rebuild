Resting

Big brass digits, legible across the gym. Nothing to read, nothing to operate — just the countdown and the set you're heading into.

2

Between sets

Tap the on-deck line and it opens into a log panel with the prescription pre-filled — 143 lb, 5 reps, RIR target 2–3 already selected. Hit the numbers as planned → one tap. Got 4 not 5, or bumped the load → nudge a stepper first. The countdown shrinks to a corner but keeps running.

3

Back to calm

Log & rest records the set and returns you to the clock with the next set already queued and the timer reset. The loop never leaves this screen.
