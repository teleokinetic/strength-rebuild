Clock front end — proposed design shifts

Non-live proposal. Static before/after fragments of the clock-live front end — nothing here is wired to the engine, and no existing file changes until a shift is approved.

Shift 1

Provenance states shouldn't rely on hue alone

Warn orange (#E8836B) and brass (#D9A03F) sit at nearly identical luminance, so the "left blank" and "progression earned" lines read the same at a glance — each state needs a distinct structure, not just a distinct hue.

Before
Was left blank last time. Nothing to carry forward — you’re starting this set from empty.
Progression earned. Add 5 lb — every set hit the top of the range at RIR 2+.

Both are thin solid left-borders differing only in hue — squint (or desaturate) and they are the same line.

After
—Was left blank last time. Nothing to carry forward — you’re starting this set from empty.
Progression earned. Add 5 lb — every set hit the top of the range at RIR 2+.

Blank: dashed edge + leading em-dash badge. Progression: solid brass edge + brass-wash fill. The two states differ even in grayscale.

Shift 2

Dashed brass means two different things

The on-deck card uses a dashed border for "tappable" while the RIR target chip uses a dashed brass ring for "target" — and once a different chip is selected, the dashed-brass ring next to the solid-brass selection reads like two selections.

Before
Left in the tank · RIR (target 2–3)

User picked 0–1, but the dashed-brass ring on 2–3 still glows beside it — which one is chosen?

After
Left in the tank · RIR (target 2–3)

Once any chip is selected, the target ring demotes to a plain border with a tiny lowercase target caption — only the selected chip reads as chosen.

Shift 3

The blank-set warning is invisible until you tap in

On the calm rest screen a blanked movement's detail sits in plain dim text, identical to a normal set — the one state that needs attention should flag itself before the log panel opens.

Before
On deck · Set 2 of 4
Front squat
left blank last time — nothing to pre-fill
tap to log ▸

The blank-set note wears the same dim gray as every ordinary "143 lb × 5" detail line — nothing asks for attention.

After
On deck · Set 2 of 4
Front squat
left blank last time — nothing to pre-fill
tap to log ▸

A 3px warn left edge plus warn-tinted detail text flags the blank set from the rest screen, before the tap.
